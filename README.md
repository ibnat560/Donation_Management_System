There are different donation sectors managed by the admins and the donors can donate under these sectors. Admins can arrange campaigns in which volunteers can participate. Volunteers use the donations in the campaigns they participate. Admins can manage other admins and volunteers whereas donors are managed by the volunteers. Donors can also calculate zakat and see self-donation history. They give feedback which are reviewed by the admins and volunteers. 
Each user can login, logout and update their own profile. Donors can become the part of the system either through signing up himself or being added by a volunteer. Admins and volunteers are added to the system by an admin. The first admin is added manually in the database.
